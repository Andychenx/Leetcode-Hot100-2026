# 283. 移动零

## 题目描述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**要求：**
- 必须 **在原地** 修改数组，不能复制数组。
- 尽量减少操作次数。

**示例：**

```
输入: nums = [0, 1, 0, 3, 12]
输出: [1, 3, 12, 0, 0]
```

## 思路分析

### 核心思想：双指针（快慢指针）

用两个指针：`slow` 指向当前 **待填入非零元素的位置**，`fast` 遍历整个数组。

- **`slow`**：下一个非零元素应该放的位置（即当前已经处理好的非零元素的尾部）
- **`fast`**：当前遍历到的元素

遍历过程中，每当 `fast` 遇到非零元素，就把它交换（或赋值）到 `slow` 位置，然后 `slow++`。

这样遍历结束后，`slow` 之前的位置都是非零元素（且保持原顺序），`slow` 之后的位置全部置零即可。

### 算法步骤（交换法）

1. 初始化 `slow = 0`
2. 遍历 `fast` 从 `0` 到 `len(nums) - 1`：
   - 如果 `nums[fast] != 0`：交换 `nums[slow]` 与 `nums[fast]`，`slow++`
3. 最终所有非零元素都移到了前面，零自然移到了后面

### 另一种理解方式

可以看作是一个 **"去零压缩"** 过程：
1. 把所有非零元素按顺序放到数组前面（覆盖写入）
2. 剩余位置补零

但交换法的好处是 **一次遍历即可完成**，代码更简洁。

## Python 代码

### 方法一：交换法（推荐）

```python
def moveZeroes(nums: list[int]) -> None:
    """原地修改，不返回任何值"""
    slow = 0  # 下一个非零元素的位置

    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow], nums[fast] = nums[fast], nums[slow]
            slow += 1
```

### 方法二：覆盖补零法

```python
def moveZeroes(nums: list[int]) -> None:
    # 1. 将所有非零元素移到前面
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow] = nums[fast]
            slow += 1

    # 2. 剩余位置补零
    for i in range(slow, len(nums)):
        nums[i] = 0
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 交换法 | **O(n)** | **O(1)** |
| 覆盖补零法 | **O(n)** | **O(1)** |

- 一次遍历完成，交换法在非零元素较多时交换次数略多，但仍是 O(n)。
- 只使用了常数额外空间。

## 易错点

1. **要维持非零元素的相对顺序** — 不能用单次排序或分区算法，否则会打乱顺序。
2. **必须原地修改** — 不能 `return [x for x in nums if x != 0] + [0] * nums.count(0)`。
3. **交换法注意 `slow == fast` 的情况**：当数组开头没有零时（如 `[1,2,3]`），`slow` 和 `fast` 同步移动，自己交换自己，没有性能问题，可以加一个 `if slow != fast` 判断来避免，但不是必须的。
4. **`fast` 遍历完整数组** — 不能提前 break，否则无法处理后面的非零元素。
