给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i`天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。

**方法**
维持一个单调递减的栈，当当前温度大于栈顶元素时，说明栈顶元素的下一最高温度找到，跟新答案数组，然后将栈顶元素pop掉，再将新的栈顶元素与当前温度比较，如果栈顶元素小于当前温度，重复先前步骤，当不小于当前温度，则将当前元素入栈。

**代码**
```
def dailyTemperature(self, temperatures: List[int]) -> List[int]:
	n = len(temperatures)
	ans = [0] * 0
	stack = []
	
	for i in range(n):
		while stack and temperateures[i] > temperatures[stack[-1]]:
			top = stack.pop()
			ans[top] = i - top
		stack.append(i)
		
	return ans

```
