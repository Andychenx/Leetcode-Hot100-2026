# 49. 字母异位词分组

## 题目描述

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 是由重新排列源单词的所有字母得到的一个新单词。

**示例：**

```
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

## 思路分析

### 核心思想
两个字符串是字母异位词 $\iff$ 它们包含的 **每个字符的出现次数** 完全相同。

因此，我们可以为每组异位词找到一个 **唯一标识**（key），将拥有相同 key 的字符串分到同一组。

### 两种方案

#### 方案一：排序法（最直观）
对每个字符串的字符排序，排序后的结果作为 key。

- `"eat"` 排序 $\rightarrow$ `"aet"`
- `"tea"` 排序 $\rightarrow$ `"aet"` $\Rightarrow$ 同一组

> 时间复杂度 O(n·k log k)，其中 n 是字符串个数，k 是字符串最大长度。

#### 方案二：计数法（更高效）
由于本题限定小写字母，可以用一个长度为 26 的数组统计每个字符出现次数，将其转为 **元组** 作为 key。

- `"eat"` $\rightarrow$ `(1, 0, 0, 0, 1, ..., 1)`（e×1, a×1, t×1）
- `"tea"` $\rightarrow$ 同样的元组 $\Rightarrow$ 同一组

> 时间复杂度 O(n·k)，比排序法少一个 log 因子。

## Python 代码

### 排序法

```python
from collections import defaultdict

def groupAnagrams(strs: list[str]) -> list[list[str]]:
    groups = defaultdict(list)

    for s in strs:
        key = "".join(sorted(s))
        groups[key].append(s)

    return list(groups.values())
```

### 计数法（推荐）

```python
from collections import defaultdict

def groupAnagrams(strs: list[str]) -> list[list[str]]:
    groups = defaultdict(list)

    for s in strs:
        count = [0] * 26          # 26 个小写字母
        for ch in s:
            count[ord(ch) - ord("a")] += 1
        key = tuple(count)        # 转为不可变的元组作为 key
        groups[key].append(s)

    return list(groups.values())
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 排序法 | O(n·k log k) | O(n·k) |
| 计数法 | **O(n·k)** | O(n·k) |

- n：字符串个数
- k：字符串的最大长度
