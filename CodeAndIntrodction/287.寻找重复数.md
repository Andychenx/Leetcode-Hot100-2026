给定一个包含 `n + 1` 个整数的数组 `nums` ，其数字都在 `[1, n]` 范围内（包括 `1` 和 `n`），可知至少存在一个重复的整数。

假设 `nums` 只有 **一个重复的整数** ，返回 **这个重复的数** 。

你设计的解决方案必须 **不修改** 数组 `nums` 且只用常量级 `O(1)` 的额外空间。

**方法**
本质是找链表环的入口，用快慢指针，当快指针和慢指针相遇，说明存在环。
再将慢指针重制，当指针相等时，则为相同数。

**代码**
```
def findDuplicate(nums: List[int]) -> int:
    slow = fast = 0
    # 第一次相遇
    while True:
        slow = nums[slow]
        fast = nums[nums[fast]]
        if slow == fast:
            break
    
    # 找环入口
    slow = 0
    while slow != fast:
        slow = nums[slow]
        fast = nums[fast]
    
    return slow
```
