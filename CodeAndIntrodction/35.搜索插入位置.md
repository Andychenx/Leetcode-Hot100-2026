给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。

**方法**
二分查找，定义左右指针，当左右指针最中间的数`nums[mid] == target`，返回`mid`
如果`nums[mid] > target`， 将左指针重置为`right = mid + 1`
如果`nums[mid] < target`， 将右指针重置为`left = mid - 1`

**代码**
```
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        
        # 循环结束时 left = right + 1
        # left 就是 target 应该插入的位置
        return left
```
