给你一个未排序的整数数组 `nums` ，请你找出其中没有出现的最小的正整数。

请你实现时间复杂度为 `O(n)` 并且只使用常数级别额外空间的解决方案。

**方法**
哈希表，在原数组上，当前元素是否满足`nums[num[i] - 1] == nums[i]`，即当前元素是否为下标元素加一。把每个元素`x`都放到下标为`x - 1`的位置上。最后便利数组，第一个位置`nums[i] != i + 1`的位置需要填入的数，即为需要补入的数。当没找到是，返回`n + 1`。


当前我们站在 `i` 这个位置，`nums[i]` 存着一个值。
- `nums[i]` 表示当前数字
    
- `nums[i] - 1` 是这个数字应该去的目标下标
    
- `nums[nums[i] - 1]` 是目标下标现在存的数字

**代码**
```
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)
        
        # 第一步：把每个数字放到正确的位置
        for i in range(n):
            # 注意：使用 while 循环，不断交换直到当前位置是正确数字或者不需要交换
            while 1 <= nums[i] <= n and nums[nums[i] - 1] != nums[i]:
                # 将 nums[i] 放到下标 nums[i]-1 的位置
                idx = nums[i] - 1
                nums[i], nums[idx] = nums[idx], nums[i]
        
        # 第二步：找第一个缺失的正数
        for i in range(n):
            if nums[i] != i + 1:
                return i + 1
        
        return n + 1
```
