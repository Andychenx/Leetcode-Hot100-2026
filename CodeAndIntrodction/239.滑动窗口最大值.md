给你一个整数数组nums，有一个大小为k的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到滑动窗口内的k个数字。滑动窗口每次只向右移动一位。
返回滑动窗口中的最大值。

**单调队列**
```
class Solution:
	def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]
		dq = deque() # (存储下标)双端队列维护一个递减队列，队首永远是当前窗口的最大值。
		res = []
		
		for i, num in enumerate(nums):
			# 移除队尾小于当前元素值的元素
			while dq and nums[dq[-1]] < num:
				dq.pop()
				
			dq.append(i)
			
			if dq[0] <= i - k:
				dq.popleft()
				
			if i >= k - 1:
				res.append(nums[dq[0]])
				
		return res
```

思路：
用双端队列（deque）维护一个递减队列，队首永远是当前窗口的最大值。
规则：
  1.队列中存的是下标，而不是值（方便判断是否划出窗口）
  2.当新元素 nums[i] 进入窗口时：
    - 从队尾移除所有小于 nums[i] 的元素（他们不可能是当前或未来窗口的最大值）
    - 将i入队
 3.移除窗口左边滑出去的元素：如果队首下标 <= i - k，就弹出队首
 4.当 i >= k - 1 时，队首对应的值就是当前窗口的最大值

复杂度分析：
 - 时间复杂度：O(n)--每个元素最多入队、出队一次
 - 空间复杂度：O(k)--队列最多存储k个元素
