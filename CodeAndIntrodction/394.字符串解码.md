题目描述：
给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。

测试用例保证输出的长度不会超过 `105`。

**方法**
使用两个栈，一个栈存储数字，一个栈存储当前累计的字符串
`stack`:存储当前累计的字符串
`count_stack`:存储字符串出现次数

遍历字符串s
当前字符为数字的时候，将数字累加起来`k=k*10+int(ch)`
当前字符为`[`时，将当前累加的字符串入栈，并将当前的累计次数入栈
当前字符串为`]`时，累加字符串等于当前累加字符串加上`count_stack`栈顶元素的`stack.pop()`次
当前字符串为字母时，累加字符串累加

**代码**
```
def decodeString(self, s: str) -> str:
	res = ""
	stack = []
	count_stack = []
	k = 0
	
	for ch in s:
		if ch.isdigit():
			k = k * 10 + k
		elif ch == '[':
			stack.append(k)
			count_stack.append(res)
			res = ""
			k = 0
		elif ch == ']':
			pre_str = count_stack.pop()
			repeat_times = stack.pop()
			res = pre_str + res * repeat_times
		else:
			res += ch
	return res
	
```
