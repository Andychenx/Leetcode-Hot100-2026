**技巧**
数组中只有0、1、2，要完成排序，使用**双指针**。
定义一个左右指针，遍历数组，当数组元素为0时，则将其交换到left指针的位置，并将left指针右移；当数组元素是1时，继续遍历；当数组元素为2时，将元素交换到right位置，并将right指针左移。

```
void sortColors(vector<int>& nums) {
	int i = 0;
	int left = 0;
	int right = nums.size() - 1;
	while(i < right) {
		if (nums[i] == 2) {
			swap(nums[i], nums[right]);
			right--;
		} else if (nums[i] == 0) {
			swap(nums[i], nums[left]);
			left++;
			i++;
		} else {
			i++;
		}
	}
}
```
